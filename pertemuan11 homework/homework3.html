FAQ Interaktif (jQuery)

1. Apa itu CSS?
CSS digunakan untuk memperindah tampilan halaman web.
2. Apa itu HTML?
HyperText Markup Language adalah bahasa standar yang digunakan untuk membuat struktur dan konten halaman web.
3. Apa itu JavaScript?
JavaScript adalah bahasa untuk membuat halaman web interaktif.
4. Apa itu jQuery?
jQuery adalah library JavaScript untuk mempermudah manipulasi DOM.
5. Apa itu Streamlit?
Streamlit adalah framework sumber terbuka berbasis python yang memudahkan untuk membuat aplikasi web interaktif dengan cepat hanya menulis kode python.
